2 PC direct connection using cable

Instruction

Preparation

Connect 2 PC using cables. Make sure the cable is reliable. Use Cat-5e or Cat-6 cable if possible.

Assume both computer run Windows system.

For PC-1

Go to Control Panel/Network and Internet/Network and Sharing Center
On the left of the current page, go to "Change adapter settings"
right click Ethernet, go to property
double click TCP/IPv4
select use the following ip address. fill in
ip address: 192.168.0.1
subnet mask: 255.255.255.0
default gate way: 192.168.0.1
now you are good for network setting
Next to to windows defender setting, turn off all firewalls
remember what you did, you need to turn on the firewall to gain protection after you finished direct connect
to transfer file, you may either use windows file sharing feature(which i do not recommend), or use a ftp server. For more information about how to set a ftp server on your computer, please use filezilla ftp server, it's very simple to set up

For PC-2

go to IPv4 setting, fill in
ip address: 192.168.0.2
subnet mask: 255.255.255.0
default gate way: 192.168.0.1
assume you have ftp server active on PC-1, you may access the file on PC-1 like the following:
return to desktop, go to this pc. on the left list you will find this pc, right click it and go to add network location. Then you follow the instructions to get access to ftp server.
